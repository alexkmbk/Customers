Учебный проект на платформе ASP .NET MVC 6 
База данных: PostgreeSQL
ORM: EF7, Dapper
frontend: jquery,jquery-ui

Функционал
Реализована возможность ведения списка контрагентов.
Доступны операции добавления\изменения\удаления.
Работа с записями ведется в отдельном диалоге с использованием асинхронных запросов.
Список хранится в виде двух связанных таблиц Customer и BusinessType
