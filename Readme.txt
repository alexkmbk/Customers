Учебный проект на платформе ASP .NET MVC 6 
База данных: PostgreeSQL
ORM: EF7, Dapper
frontend: jquery,jquery-ui, bootstrap(для оформления таблицы)

Функционал
Реализована возможность ведения списка контрагентов.
Доступны операции добавления\изменения\удаления.
Работа с записями ведется в отдельном диалоге с использованием асинхронных запросов.
Список хранится в таблице Customer
Дополнительные свойства хранятся в связанных таблицах:
BusinessType - типы контрагента (физ.лицо\юр.лицо)
BankAccounts - банковские счета контрагента
Banks - банки

Список банков в таблицу Banks можно загрузить из Excel файла.
